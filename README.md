# atm-machine

#overview :  This project is a simulation of interaction of atm machine it contains details of 5 bank account holders in its database they can use there account number and pin to withdraw               or deposit money or check bank balance and reset pin.    

#features :   1) It contains database of 5 bank account holders and their pins.
              2) We can Withdraul the money.
              3) We can Deposit the money.
              4) We can Check Balance.
              5) We can Reset the pin.
              
#technologies/tools-used : 1) time function.
                           2) if and if else statements.
                           3) def function.
                           4) loop statements.
                           5) print function.
                           6) input function.

#steps_to_install_and_run_the_function : 1) Download the code from Github.
                                         2) Run atm-machine.py

#instruction_for_testing : 1) After running the code it asks to enter the account number which will be in the database in the codes.
                           2) After entering the account number it asks to enter the pin.
                           3) After that it asks to enter the number from 1 to 5 which menas
                             a) "Account Balance"
                             b) "Deposit"
                             c) "Withdrawl"
                             d) "Reset Pin"
                             e) "Exit"
                           4) After entering the choice it shows the data from the database.
